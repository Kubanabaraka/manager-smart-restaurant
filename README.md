# MANAGER SMART RESTAURANT

React + Tailwind CSS manager console for a smart restaurant system. This project focuses on UI only using mock data (no real API calls).

## Tech stack

- React (functional components)
- Vite-style build setup
- Tailwind CSS
- React Router

## Available pages

- Manager Login
- Dashboard (today's orders, revenue, top items, quick actions)
- Menu Management
  - Menu List
  - Add Menu Item
  - Edit Menu Item
- Waiter Management
  - Waiter List
  - Add Waiter
  - Edit Waiter
- QR Code Generator
- Settings

## Run locally

Install dependencies and start dev server:

```bash
cd .
npm install
npm run dev
```

Then open the printed local URL in your browser.
